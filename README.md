# Introduction to TypeScript — Practical assignment

A simple web application written in **TypeScript** using **Tailwind CSS**.

## What's implemented:
- Enum and iterating through its values
- Typed variables and a function with conditional logic
- Working with DOM
- Entering name, age, hobbies, and experience

## Startup
```bash
git clone https://github.com/My-Nickname/TypescriptPractice.git
cd TypescriptPractice
npm install
npm run dev
```

Open index.html in your browser — everything works instantly!
## Technologies
- TypeScript
- Tailwind CSS v3 (CDN)
- Vanilla JS
